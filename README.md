# Mortal Kombat Themed Babbage Puzzle

## Overview
This project is a visually stunning, interactive web application inspired by Mortal Kombat, designed to solve a classic number theory problem known as the "Babbage problem." The goal is to find the smallest positive integer whose square ends with specific digits, showcasing advanced JavaScript techniques and a captivating UI that will impress employers and fellow developers.

---

## Features
- **Immersive Mortal Kombat style design** with neon glow and dark themes.
- **User input fields** for customizing the target last digits and pattern digits.
- **Efficient algorithm** to find the smallest integer satisfying the pattern.
- **Responsive layout** for desktop and mobile devices.
- **Clean separation** of HTML, CSS, and JavaScript for maintainability.

---

## Technologies Used
- HTML5
- CSS3 (including Google Fonts)
- Modern JavaScript (ES6+)
- No external dependencies aside from Google Fonts

---

## How to Use
1. Clone or download this repository.
2. Open `index.html` in your preferred web browser.
3. Enter the target last digits (default: 269696) and pattern digits.
4. Click **"Start the Kombat"** to find the smallest number that satisfies the condition.
5. The result will be displayed with a stylish Mortal Kombat effect.
